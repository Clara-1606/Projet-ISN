Comment jouer ?

Pour accéder au jeu, il suffit de le télécharger avec le lien juste en dessous.

Pour jouer, il vous suffit de cliquer plusieurs fois sur les boutons tout en bas afin de choisir les billes de couleurs que vous voulez, vous pouvez cliquer sur RESET pour reinitialiser vos choix.
Une fois choisis, cliquer sur valider, si des couleurs d'affiches, il est à la bonne place ! S'il reste gris, alors soit la couleur n'est pas à la bonne place ou alors elle n'est pas dans le code !
Vous avez 10 essais pour deviner le code gagnant, et vous pouvez rejouer autant de fois que vous voulez en cliquant sur REJOUER.
